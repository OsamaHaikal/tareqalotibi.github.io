    Smart Parking Management System


    In my graduation project, I created a prototype for a Smart Parking Management System to enhance and simplify the parking process for drivers.
    This innovative project utilizes IoT technology and intelligent algorithms to track real-time parking availability, helping users locate empty spaces efficiently. Additionally, the system allows users to reserve parking spaces and process payments via a user-friendly website. By combining hardware components (Raspberry Pi 3 Model B, ESP32, IR Sensor, SG-90 Servo Motor, etc.) with software solutions by using the Node-Red technology, which is a programming tool used to connect the hardware devices with the software services.
    The Smart Parking Management System contributes to reducing traffic congestion, Protecting property, saving time, and improving overall parking management in urban areas.
